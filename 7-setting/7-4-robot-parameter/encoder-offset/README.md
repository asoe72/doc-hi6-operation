# 7.4.4 엔코더 옵셋

현재 엔코더의 위치를 엔코더의 원점 위치\(0X400000위치\)로 설정할 수 있습니다. 로봇 각 축의 기준 위치\(각 축의 스케일이 부착된 위치\)에서 엔코더의 원점을 결정합니다.

1. \[3: 로봇 파라미터 &gt; 4: 엔코더 옵셋\] 메뉴를 터치하십시오.
2. 각 축의 위치를 조정하여 엔코더 옵셋값을 설정하십시오. 엔코더 옵셋값은 헥사값\(Hexa값, 16진수\)으로 기록됩니다.
